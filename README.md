# Toposel Assignment – Custom WordPress Theme

## 📌 Project Overview

This project is a custom WordPress theme developed as part of the Toposel assignment.
The goal was to create a dynamic homepage using WordPress and Advanced Custom Fields (ACF) without hardcoding any content.

---

## 🚀 Features

* Custom WordPress theme (built from scratch)
* Dynamic homepage using ACF (Advanced Custom Fields)
* No hardcoded content – fully editable from WordPress admin
* Mobile-first layout (as per Figma requirement)
* Clean and simple PHP structure

---

## 🧩 Sections Implemented

* Announcement Bar
* Hero Section (Image, Heading, Subheading, Button)
* Brand Logos
* Product Section (Image, Name, Price)

---

## 🛠️ Tech Stack

* WordPress
* PHP
* Advanced Custom Fields (ACF)
* HTML/CSS

---

## ⚙️ Setup Instructions

1. Install WordPress locally (using Local WP or XAMPP)
2. Copy the `custom-theme` folder into:

   ```
   wp-content/themes/
   ```
3. Login to WordPress Admin
4. Go to **Appearance → Themes** and activate **Custom Theme**
5. Install and activate **Advanced Custom Fields (ACF)** plugin
6. Create required fields (as per project structure)
7. Add content from **Pages → Home**
8. View the homepage
